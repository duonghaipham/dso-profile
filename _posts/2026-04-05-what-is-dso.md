A DSO (Digital Skill Online) profile is a way of making your technical skills and portfolio visible and verifiable online. Think of it as a living resume — one that grows as you do.

## Static Resume vs. Living Profile

A traditional resume is a snapshot frozen in time. It lists where you've been, but says little about what you can do *today*. A digital profile is different:

- It links to actual code (GitHub)
- It includes real project outputs
- It updates continuously as you build and learn

## What Goes Into a Good Profile

1. **Projects** — concrete examples of things you've built
2. **Skills** — honestly assessed, not padded
3. **Writing** — blog posts or documentation show how you think
4. **Contact** — make it easy for the right people to reach you

This site is my attempt at building exactly that.
